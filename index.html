litegapps is a custom Gapps (Google Apps) on the Android operating system based on Google Pixel devices, an open source project that focuses on efficiency, friendliness and completeness. This project was first developed by Wahyu6070 in 2020, the first release supports Android 5.0 (Lollipop) up to the latest version (Android 14) with support for various devices.

We offer solutions for the Android community in GAPPS (Google Apps) which can handle all user cases regarding GAPPS, many of our products can be used to handle all user cases. we develop what is lacking from gapps so far and then provide answers to complaints so far.

Installer

Litegapps uses a unique installer, namely Kopi Installer, which is able to handle various installation locations on Android and several user cases. Let's learn about the supported installer support:

RECOVERY

You can install via custom recovery such as TWRP, OrangeFox Recovery, Lineage Recovery or others. This will be installed in the Android partition, we call it non systemless

Name prefix RECOVERY_litegapps_xxxxx.zip

MAGISK

Litegapps also supports installation via Magisk which means it will install as a Magisk module (systemless), make sure you use Magisk 20.4+

Name prefix MAGISK_litegapps_xxxxx.zip

KernelSU

not to be left behind with the latest Android root application, we also added support for installing kernelsu, this will be installed as a kernelsu module (systemless)

Name prefix KSU_litegapps_xxxxx.zip

AUTO

with a copy of the AUTO version of the installer, you can install in various places such as RECOVERY,MAGISK,KSU which can be converted to systemless or non-systemless automatically/manually. For further instructions, you can read the documentation

Name prefix AUTO_litegapps_xxxxxx.zip

SIDELOAD

litegapps supports installation via adb sideload on mobile/desktop and below is an example code:

adb sideload (name litegapps zip file)

Variant

Pixel

All Google Apps components from the latest Google Pixel series devices are included to be ported to this variant, so you will enjoy the look and apps like on Pixel devices in your ROM. However, it has a large size of around 600MB - 1GB So it will take up a lot of storage and RAM memory, we do not recommend it for devices that have small RAM and internal storage.

Micro

The component of the micro is taken half of the pixel variant, so there are fewer Google apps and takes up less memory, it has a size of around 300MB - 600MB,. This variant is still considered heavy for devices with small memory, so consider it accordingly your device needs and specifications.

Nano

This variant takes some of the components of the micro, so it has fewer Google Apps components, has a size of around 150MB - 300MB. This variant is lighter than Pixel or micro We recommend it for devices that use small internal memory and RAM.

Basic

Some of the components of the nano are taken, which are important, which have the basic functions needed, it has a small size but still maintains the good function of Google Apps. It has a size of around 150MB - 200MB we recommend for devices that have memory small RAM or low specifications.

User

All components of Google Apps are taken from users' needs by voting in the Telegram group.

Go

This variant is intended for devices with low memory or low specifications, in its components it uses Google Apps Go, for example Gmail Go, Google Search Go, Maps Go and others. This means that this variant uses a small size. but you can still use the main Google application.

Core

all the main components of Google Apps have been trimmed until the main components remain, this has a minimum size and apps (60MB - 80MB), this variant is the second most downloaded after the lite variant

Lite

the smallest and minimalist variant, only the core components of Google Apps, namely Google Play Services, Playstore, Google Services framework and Common. This is very light but there are lots of Google apps available so you can Download the Google apps that you like in the Play Store.

LiteGapps++

LiteGapps double plus is a development of regular Litegapps which is able to support various versions of Android and multiple architectures with just one flashable.zip while still maintaining a small size. but we do not provide variants in Litegapps++.

LiteGapps++ only has the same core Google Apps components as litegapps variant lite, so it focuses on minimalism. Litegapps++ is divided into three types, namely:

reguler

LiteGapps++ Regular is updated regularly using the latest version of Playstore and Google Play Services, but it only supports Android 9 (Pie) to Android 14.

LTS

LiteGapps LTS is intended for users who rarely like to update regularly, so it is designed for the long term without having to update regularly. It has extensive support, namely Android 5.0 (Lollipop) up to Android 14, but has the disadvantage of using the Playstore and Google Play versions. long service.

MicroG

LiteGapps++ MicroG uses MicroG as the default gms, it has extensive support for Android 5.0 (lollipop) up to the latest Android 14 and is very light. But many of the main functionalities don't work, such as: payments, connecting accounts to games and many others.

Addon

LiteGapps Addon is an additional package that we provide to suit user needs, users can download which package they like and then use it, to complete the LiteGapps package that doesn't have a complete package.

It also supports installation via Magisk module, KSU module, recovery, adb sideload, litegapps controller and you can even do so without installing the main litegapps package.

LiteGapps Remover

Litegapps Remover is a tool for debloating Google Apps on Android Roms, to overcome Google Apps that are too many and heavy on the default Rom./p>

Supports systemless and non-systemless installations, it is recommended to install LiteGapps Remover before installing Litegapps for ROMs that already include Google Apps

Systemless

Litegapps Remover supports installation as a Magisk module or KSU module, which means you can debloat it systemlessly and restore the debloated application completely.

Non systemless

Litegapps remover can also be installed as recovery (non systemless), this will be permanently deleted on the partition.

LiteGapps Controller

litegaps controller is an additional feature of litegaps and litegaps++, which has many features and tweaks for the user experience in using litegaps on Android devices.

For complete instructions on how to use it, you can read the documentation and below are some of the LiteApps controller features:

Boost

This boosts your device, by forcing applications to stop in the background, this will also make CPU and GPU performance lighter due to the load from applications that are not in use.

Gaming Mode

mode intended for game use to make it faster when playing games.

Battery Saver

Save battery by force stopping some applications in the background and reducing CPU and GPU performance.

Device Info

information on your device's software and hardware.
