Installation Methods

LiteGapps offers two installation modes: systemless and non-systemless.

Systemless

Install securely as a module via Magisk, KernelSU, or APatch for easy, conflict-free management and removal.

Non-Systemless

Supports traditional installation through custom recovery for permanent, deeply rooted integration into the system partition.

Sideload

Install effortlessly via adb sideload. Here is an example command for your terminal:

adb sideload litegapps-filename.zip

Available Variants

Pixel

Includes all Google Apps from the latest Pixel devices. Large size (600MB - 1GB). Not recommended for devices with limited storage/RAM.

Micro

A smaller subset of the Pixel variant (300MB - 600MB). Still may be heavily demanding for low-spec devices.

Nano

A lighter version with fewer components (150MB - 300MB). Highly recommended for devices with average memory capacity.

Basic

Contains only essential components while maintaining core functionality (150MB - 200MB). Ideal for low-spec devices.

User

A community-driven customized package where app selection is based purely on user votes from our Telegram group.

Go

Designed explicitly for low-memory devices, featuring Google's lightweight "Go" edition apps (e.g., Maps Go, Gmail Go).

Core

A minimal, barebones package with only the most crucial components (60MB - 80MB). One of our absolute most popular variants.

Lite

The smallest, most minimalist variant. Contains only Google Play Services, Play Store, and GSF. Perfect for users who want maximum control.

LiteGappsX

An advanced evolution of LiteGapps, supporting multiple Android versions and architectures directly in a single, small flashable zip. Ideal for power users who frequently switch ROMs. Three distinct types available:

  • Regular: Updated with the latest Play Services, thoroughly supporting Android 9-14.
  • MicroG: Uses MicroG as the GMS core for extensive Android 5.0+ support, though note that some functionalities like advanced payments might not work natively.

LiteGapps Controller

An incredibly powerful companion app designed for managing LiteGapps features dynamically. It provides numerous under-the-hood tweaks to enhance your overall user experience seamlessly. Please check the detailed documentation for a full, comprehensive feature list.

Explore Controller Features

Addons

Highly optional, modular packages to strictly supplement your LiteGapps installation. You can intuitively install exactly what you need via Magisk, recovery, or the integrated Controller app, even without having the main LiteGapps package installed beforehand.

LiteGapps Remover

A highly effective technical tool specifically built for safely debloating pre-installed Google Apps directly from your ROM partition. Highly recommended for use precisely before installing LiteGapps on ROMs that already include persistent GApps.

Tips And Tricks

  • For low-spec hardware devices, we strongly recommend using the Core or Lite variants.
  • Before initiating any installation, it is considered a very good technical practice to manually uninstall all updates for existing Google apps.
  • LiteGapps inherently includes a sophisticated GMS Doze feature to automatically and efficiently save battery life whenever the device is completely idle.
  • Utilize the command terminal to manually activate deep performance boosts immediately before gaming sessions, or rapidly enable the extreme battery saver when battery power is critically low. See the official docs for exact commands.
