So that we know clearly what your problem is, there are several ways to report your problem.

Send log

LiteGapps logs during install mode

1. First open file manager

2. Go to the directory /sdcard/Android/litegapps

3. You will find a file named with the prefix [LOG]litegappps_version.tar.gz example [LOG]litegappps_v2.5.tar.gz

4. It is log from litegapps which contains your device information and installation process

5. Send log in Telegram or Github

Optimal

So that we can find out more about your problem

• Send logs from recovery or magisk manager

• Send screenshot or video
