This page contains some general user questions that we answered.

What are the risks of installing LiteApps?

If you install LiteApps you will not run into serious risks, that’s because LiteApps has a safe system when installing. If your device is not compatible then LiteApps will refuse to install.

Litegapps supports what version of Android?

Litegapps has a wide and complete version of Android, supporting Android 5.0 Lollipop up to the latest release of Android 15, and we will quickly respond to updates from Google.

What are the components of LiteApps?

Litegapps has components that vary depending on which package you choose and which product you like, but we are quite complete in giving users choices.

does litegaps have any bugs or serious problems

Every release of LiteApps has been tested many times so that every version we release to the public is stable and ready to be used by users, but that doesn’t rule out the possibility of not having problems, there may be minor problems, but we are open to receiving all problem reports. found by users.

What is LiteApps variant?

Litegapps has 9 quite complete variants that can be selected according to user needs.

Does LiteApps support Chinese MIUI ROM?

Litegapps supports almost all ROMs including Chinese MIUI ROMs, but you have to install the special MIUI package version that we provide.

What is the size of LiteApps?

Litegapps has a size that varies according to the version or product chosen, but for the smallest size, LiteApps, the Lite version is around 46MB, up to the largest size, LiteApps, the pixel variant is 900MB, but this size can change at any time.

What architectures does LiteApps support

Litegapps has broad architecture support arm 32 bit, arm64 64bit, x86 32 bit and x86_64 64 bit.

what is the difference between litegaps and litegaps++

litegaps and litegaps++ have striking differences, for example litegaps++ has a cross install system and regular litegaps has a single install system. For further explanation, you can read the documentation.

does litegaps support magisk module

Yes, LiteApps fully supports the Magisk module and its system, even since its initial release in 2020, Litegapps has supported the Magisk module and not only that, we now also support the KSU module.

who is the main developer of litegaps

Litegapps was developed by a senior developer in the Android community, namely Wahyu6070 @xda-developers

does litegaps support addons

Litegapps supports quite complete addons ported directly from Google Pixel devices, these can be installed anywhere in Magisk KSU or Recovery.

What makes LiteApps different from other GAPPS?

Litegapps has many differences from other Google apps, for example:

  1. Litegapps supports systemless and non-systemless installations.
  2. Litegapps supports old to latest versions of Android.
  3. Litegapps has an addon ported from Google Pixel, and is updated actively
  4. Litegapps has many variants that users can choose from
  5. Litegapps has an active community and developers.
